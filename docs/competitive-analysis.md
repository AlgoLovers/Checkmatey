# 경쟁 분석 & 방향 — Checkmatey

기존 체스 앱을 분석해 Checkmatey가 **어디서 이길 수 있는지**와 나아갈 방향을 정한다.

## 시장 지형

| 앱 | 강점(해자) | 약점/빈틈 |
|---|---|---|
| **Chess.com** | 압도적 콘텐츠(레슨/퍼즐/봇/대전), 커뮤니티, 코치(Coach) 분석 | 온라인 필수·계정 필수, 유료 벽, 초보에겐 과함, 무겁고 광고 |
| **Lichess** | 무료·오픈소스, 퍼즐/분석/오프닝 익스플로러, **CC0 데이터** | 온라인 중심, 학습 가이드는 약함(스스로 찾아야), 초보 온보딩 약함 |
| **Chessable** | 간격 반복(MoveTrainer)로 오프닝/전술 암기 최적화 | 콘텐츠 대부분 유료, 대국/실시간 코치 약함 |
| **Dr. Wolf / Chess.com Coach** | 대화형 코치, 초보 친화 설명 | 유료·온라인, 폐쇄적 |
| **Aimchess / DecodeChess** | 약점 분석·설명(내 게임 리뷰) | 온라인·계정·구독, 대국/훈련은 외부 의존 |
| **Magnus Trainer** | 게이미피케이션 학습 | 콘텐츠 폭 제한, 구독 |

**공통 관찰**: 최상위는 콘텐츠·커뮤니티·온라인이 해자다 — 우리가 정면으로 못 이긴다.
하지만 **거의 모두 온라인·계정·구독**을 요구하고, **초보의 "왜"를 즉석에서 오프라인으로 짚어주는 경험**은 비어 있다.

## Checkmatey의 포지셔닝 (이길 수 있는 니치)

> **"오프라인 · 계정 불필요 · 코치 중심 · 레이팅 기반의, 초·중급 실력 향상 전용 앱"**

우리가 이미 가진 차별점:
- **완전 온디바이스**: 대국·퍼즐·힌트·게임 리뷰가 네트워크 없이 즉시. (비행기/지하철/데이터 걱정 X)
- **코치가 항상 붙음**: 두면서 실시간 채점, 힌트의 "왜", 게임 복기 — 무료로.
- **모든 것 레이팅 기반**: 내 Elo로 봇 난이도·퍼즐이 적응. 진행이 숫자로 보임.
- **계정·구독·광고 0**: 프라이버시·마찰 없음.

이 조합은 Chess.com/Lichess가 **의도적으로 잘 안 하는** 영역이다(그들은 온라인·커뮤니티가 본질).

## 나아갈 방향 (우선순위)

### 지금~단기 (니치에서 "최고"에 직결)
1. **약점 타겟 훈련 루프** — 게임 리뷰·퍼즐 실패에서 내 약한 테마를 뽑아 그 테마를 집중 출제 + 틀린 퍼즐 간격 반복. *(진행: 퍼즐 약점 집중/복습 큐 구현됨. 다음: 게임 리뷰의 실수 테마 → 퍼즐 연결)*
2. **콘텐츠 양** — 퍼즐 수천 개(**Lichess CC0**, 라이선스 안전). *데이터만 확보되면 즉시 연동.*
3. **체감(재미)** — 사운드(수/캡처/체크), 온보딩 튜토리얼, 데일리 목표·스트릭, 성취.

### 중기
4. **오프닝 트레이너** — 레퍼토리 드릴 + 간격 반복(Chessable식, 단 오프라인·무료).
5. **엔드게임 훈련** — 필수 엔드게임(K+Q, K+R, 대립 등) 드릴.
6. **엔진 강화** — quiescence 완료. 다음: make/unmake로 탐색 속도↑ → 더 깊은 상위 레벨, 평가바.
7. **내 게임 영속 저장** — 대국 히스토리 저장·재검토, PGN 내보내기/공유.

### 장기
8. **개인화 커리큘럼** — 내 레이팅·약점에 맞춰 "오늘 할 것"을 자동 추천(적응형 학습 경로).
9. **선택적 온라인**(니치를 지키되): 친구와 비동기 대국, 클라우드 백업. (계정은 선택)
10. **강한 분석 옵션**: Stockfish(GPL 수용 시) 또는 서버 분석 — 고급 사용자용.

## 원칙 (흔들리지 말 것)
- **오프라인·무료·프라이버시**가 정체성. 온라인/구독으로 Chess.com를 흉내내지 않는다.
- **모든 기능은 레이팅과 학습에 기여**해야 한다(재미도 학습을 강화하는 방향으로).
- 콘텐츠는 **라이선스 안전한 오픈 데이터(Lichess CC0)** 만.
